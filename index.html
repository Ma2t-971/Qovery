How does it work?

Qovery Engine is an open-source abstraction layer product that turns easy apps deployement on AWS, GCP, Azureand others Cloud providers. The Qovery Engine is coded in Rust and takes advantages of Terraform, Helm, Kubectl, Docker to manage ressources.

Initialize EKS (AWS Kubernetes) and ECR (AWS container registry) on AWS
